Key Features

    Project Overview: I deployed a basic web app on an AWS EC2 instance using Apache as the web server. The project includes setting up firewall rules, serving an HTML page, and beginning the process of enabling HTTPS.

    1. Launched an EC2 Instance on AWS:
      Created a free-tier virtual machine using Amazon Linux 2 as the cloud server.
    2. Configured Security Group Rules:
      Allowed inbound traffic on ports 22 (SSH), 80 (HTTP), and 443 (HTTPS) through AWS firewall settings.
    3. Connected to the Server Using SSH:
      Used a terminal and a .pem key file to securely access the EC2 instance.
    4. Installed Apache Web Server:
      Installed and enabled Apache (httpd) to serve web pages from the server.
    5. Created a Simple HTML Web Page:
      Wrote an index.html file and placed it in Apache web root directory.
    6. Adjusted Firewall Rules on the Server:
      Verified or configured firewalld or iptables to allow external HTTP/HTTPS traffic.
    7. Tested the Web App in a Browser:
      Entered the EC2 public IP in a browser and successfully viewed the hosted web page.
    8. Started Configuring HTTPS (In Progress):
      Initiated SSL setup using Certbot and OpenSSL to enable secure HTTPS access.
